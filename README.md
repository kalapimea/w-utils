# w-utils
w-utils is a free CLI based app inspired by the Finnish school portal app Wilma, to keep track of your classes, grades, upcoming exams and more.

# usage:
The commands found inside of w-utils should explain themselves really well, if you need a list of commands, type `help` into the terminal. If you need help with individual commands, type `[command] help`

Disclaimer: This project is completely independent and does not use the Wilma API, nor is it associated with the original product from Visma in any way.
